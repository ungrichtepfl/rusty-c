# Rusty-C

## Pre-requisites

### LLVM & Libclang

**Windows**:

```shell
winget install LLVM.LLVM
```

**Debian based**:

```shell
sudo apt install llvm-dev libclang-dev clang
```

For **other distros** checkout the bindgen [requirements page](https://rust-lang.github.io/rust-bindgen/requirements.html).
